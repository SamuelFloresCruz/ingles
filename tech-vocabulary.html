Glossary

AI Vocabulary

Technical words with definitions, images, and example sentences related to artificial intelligence.

AI

Artificial Intelligence

Definition: Technology that allows machines to learn, reason, and make decisions.

Example: Artificial intelligence can help doctors analyze medical images.

ALG

Algorithm

Definition: A set of instructions that a computer follows to solve a problem or complete a task.

Example: The algorithm recommends videos based on user preferences.

DATA

Dataset

Definition: A collection of information used to train or test an AI model.

Example: A large dataset can improve the accuracy of a machine learning model.

NN

Neural Network

Definition: An AI model inspired by the human brain that recognizes complex patterns.

Example: A neural network can identify objects in a photograph.

Vocabulary grows through practice.
